# 方法

1. Android Studioでプロジェクトを起動する
2. deviceとしてiOS Simulatorを選択する。
3. Simulatorのメニューバーから以下の順に選択する。

    File --> Open Simulator --> iOS 15.0(2021/12/09現在)

4. 使用したいdeviceを選択するとそのSimulatorが起動する。
5.  Android Studioから使用したいdeviceを選択する。
